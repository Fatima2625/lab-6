In this lab, a Linear Regression model was applied to predict the yearly amount spent by customers.
The dataset was explored using basic methods such as head(), info(), and describe().
No missing values were found, so no data cleaning was required.
The model was trained using four features: session length, time on app, time on website, and membership length.

The evaluation results show:
MAE = 7.74
RMSE = 9.68

These results indicate that the model performs well and provides accurate predictions.
The scatter plot shows a strong linear relationship between actual and predicted values, confirming good model performance.
